# sublime text

安裝package control  
[https://packagecontrol.io/](https://packagecontrol.io/)  
以下是個人使用套件

## AutoFileName

可以自動帶入資料夾檔案  
[https://packagecontrol.io/packages/AutoFileName](https://packagecontrol.io/packages/AutoFileName)

## converttoUTF8

[http://blog.developer.idv.tw/2015/07/sublime-text-3-package-converttoutf8.html](http://blog.developer.idv.tw/2015/07/sublime-text-3-package-converttoutf8.html)

## Bootstrap3 snippets

bs3快手  
[https://github.com/JasonMortonNZ/bs3-sublime-plugin](https://github.com/JasonMortonNZ/bs3-sublime-plugin)  
[https://www.youtube.com/watch?v=VWp6J7Q59KE](https://www.youtube.com/watch?v=VWp6J7Q59KE)

## BracketHighlighter

可以修改前後括號的highlight樣式  
[http://facelessuser.github.io/BracketHighlighter/](http://facelessuser.github.io/BracketHighlighter/)

> 找到"bracket\_styles": {這行  
> 把裡面的"style": "underline"  
> 改成"style": "hightlight"

## color picker

可以在sublime text裡面叫出色盤  
（如果sublime text使用全螢幕，叫出color picker似乎會當掉）  
[和converToUTF8 相衝的修改方式](http://www.bos.tw/2017/02/sublime-text-3-%E7%9A%84%E5%A4%96%E6%8E%9B-colorpicker-%E5%AE%89%E8%A3%9D%E5%BE%8C%E7%84%A1%E6%B3%95%E4%BD%BF%E7%94%A8%E4%B9%8B%E8%A7%A3%E6%B1%BA%E6%96%B9%E6%B3%95/)

## color Highlighter

程式中的色碼可以帶出顏色來  
[https://packagecontrol.io/packages/Color Highlighter](https://packagecontrol.io/packages/Color%20Highlighter)

## Emmet

前端必備，其中可以使用css選擇器快速編寫html出來，很好用  
[http://abgne.tw/web/emmet-tutorial-series-basic.html](http://abgne.tw/web/emmet-tutorial-series-basic.html)  
[https://pjchender.blogspot.tw/2016/07/emmet.html](https://pjchender.blogspot.tw/2016/07/emmet.html)  
[https://docs.emmet.io/cheat-sheet/](https://docs.emmet.io/cheat-sheet/)
