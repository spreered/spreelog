轉職工程師也接近一年了，除了埋首專案以外，也兼作公司程式轉職班的助教。雖然我在程式技術上還有很多地方要學習（每天 pr 都被改得不要不要），但從自己在 coding bootcamp（程式訓練營） 的學習過程、過往的工作經歷以及兩次轉職的經驗，加上已經輔導了兩屆的 bootcamp 學員，已經看過很多程式素人到成功轉職的案例（至少我的同學最久也都轉職成功一年半了），自己對於參加 bootcamp 整個過程和須具備的心態還蠻有心得的，因此想要寫一下這系列的文章來分享一下。

(其實還有一個原因是，公司每屆 bootcamp 學員會發生的問題大部分都是那些，不管是心理上還是技術上的，有時候重複講還蠻累的，所以先寫下來以後請大家來看一下...。)

# 簡單的自我介紹

#### 資訊工程系

我大學是資訊工程科系畢業的，在學的成績還可以，但是那時候就是不太知道自己想幹嘛，也不太甘心就這樣跟著大家考研究所然後去科技公司就這樣一輩子下去（現在想起來每次轉職都是這個念頭...）所以就跟著社團老師跑去做吉他教學。

#### 吉他老師

其實我吉他也沒彈得很好，一開始教的時候彈奏技巧和教學技巧都蠻慘的，學生一直掉。教了一陣子之後，其實也漸漸有心得了，比如說分析一些歌曲可以教學哪些主題、要怎麼樣面對不同年齡層及不同程度的學生，盡量讓學生開心學習（喜歡的歌曲）的狀況下又可以有進步。另外被老闆「磨練」下，也訓練出一些軟技能，例如行政上面的統合能力和銷售產品的技巧。雖然這個部分沒辦法真的寫在履歷上，但我覺得這段時間的成長對於我在日後工作對人際處理上都蠻有幫助的。

教了快五年，看到了音樂及樂器市場的一些瓶頸，但主要還是工作型態沒辦法跟一般人的作息配合，別說朋友，連女友和家人在這幾年都很少見到我，所以後來決定離職去考公務員。

#### 公務員

因為是資工系畢業，所以挑了報考資訊處理類科的國家考試，大概花了不到一年的時間幸運地吊車尾考上高考。說運氣也有，說考試技巧也有，但這段時間我領悟到蠻多學習上的技巧和心態，這個在後來轉職過程中也應證了這些技巧是有效的。

公務員的生活穩定，還讓我有時間玩點音樂、學底片沖放相和學日文，即使工作上游刃有餘但不太開心，主要是工作畢竟還是佔了一天醒著一半以上的時間，我又回到了「難道我要這樣一輩子下去嗎」的憂鬱迴圈，所以在最後幾年我開始回來自學寫寫程式。一開始是寫興趣，後來在小孩子出生後想要轉職的念頭越來越強，於是我就報名了 ALPHA Camp 的 coding bootcamp，最後離開了公務體系。

#### 網路工程師

寫這篇文章的時候大概已經轉職為網路工程師一年了，雖然比公務員時期忙很多，但是每天都過得很充實，很有挑戰，除了學新技術，工作的時候也常常應用到以前學到的技能比如說拍照或pa音響之類的。
